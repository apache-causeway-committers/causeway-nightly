1.16.2

This release is primarily a bug fix release fixing one severe error relating to parented checkboxes (ISIS-1874). It does however introduce one new feature for the REST API (RO viewer) to support generic RO clients. There is also one notable performance improvement through the caching of JAXBContexts.

New Feature

  • ISIS-1882 - Extend RO viewer to (optionally) automatically wrap action-results in a view model.

Improvement

  • ISIS-1884 - Restrict where choices and autoComplete are implicitly provided on parameters

  • ISIS-1879 - Date header returned by RO resuts should be the Clock time, not the system time.

  • ISIS-1877 - Enhance representation of action invocation

  • ISIS-1876 - Convert IsisApplibModule (and archetype app modules) to use ModuleAbstract

  • ISIS-1872 - Extend ModuleOrBuilderAbstract, with #withAdditionalDependencies(Module…​ modules)

  • ISIS-1867 - Avoid queuing/contention issues - cache JAXBContext in MemberExecutionDtoUtils.

Bug

  • ISIS-1874 - Parented checkboxes fails - NPEs and incorrect facets.

Task

  • ISIS-1875 - Release tasks for 1.16.2
