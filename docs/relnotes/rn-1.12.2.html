1.12.2

Apache Isis 1.12.2 is a bug fix release for Apache Isis 1.12.0. It addresses two UI issues which result in the viewer showing invalid data with respect to the data actually stored in the database.

Bug

  • ISIS-1396 - Some elements on an entity page can be rendered against stale data.

  • ISIS-1132 - possible timezone issues when refresh page in Wicket viewer.

Task

  • ISIS-1424 - Release activities for v1.12.2
