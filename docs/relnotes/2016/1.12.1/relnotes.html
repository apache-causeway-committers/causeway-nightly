1.12.1

Apache Isis 1.12.1 is a bug fix release for Apache Isis 1.12.0. The primary issue it addresses are spurious concurrency exceptions that can be thrown when editing an object. It also addresses a subtle issue where the new support in JAXB view models for auto-edit can cause some dirty objects not to be updated.

Bug

  • ISIS-1355 - concurrency exception after property edit when attempting to render…​

  • ISIS-1357 - Update dependency from guava-19.0.20150826 to guava-19.0

  • ISIS-1361 - View model that changes two domain objects only has one of them updated.

  • ISIS-1363 - ConfigurationServiceMenu is defined under the prototyping menu, but is not restricted to prototyping. Suggest place under the tertiary menu instead.

Task

  • ISIS-1362 - Release activities for v1.12.1
