1.13.1

This is a maintenance release with a number of minor improvements and bug fixes. It also provides better support for docker, by allowing configuration properties to be overridden using either system properties or an ISIS_OPT environment variable.

New Feature

  • ISIS-1510 - Improve support for docker containers by allowing isis configuration properties to be overridden using either system properties or an ISIS_OPT environment variable

Improvement

  • ISIS-1527 - Make overlays consistent for editable properties, with general fix for Firefox

  • ISIS-1522 - Remove unused field declarations

  • ISIS-1519 - Deprecate JdoColumnLength in applib, instead define types

  • ISIS-1518 - Hide and replace certain UI fragments

  • ISIS-1514 - Hide the calendar hint on the date time (users tend to click on it)

  • ISIS-1508 - The integration test framework should be more resilient to badly behaved tests that leave the transaction incomplete

  • ISIS-1507 - QueryResultsCache should be disabled while fixture scripts are running.

  • ISIS-1505 - MetaModelService should return an unknown enum rather than throw exception, eg if passed an java.lang.Object.class.

  • ISIS-1499 - Improve BookmarkService to be able to "lookup" domain services, so that can schedule background commands against a service.

  • ISIS-1495 - Extend SessionLoggingService.CausedBy enum, so can mark as complete any log entries that are still shown as ongoing when app (re)starts

  • ISIS-1492 - Adding css-class to object forms

  • ISIS-1491 - Changed all occurrences of allowNulls and allowsNulls in the docs to

  • ISIS-1476 - Transaction interface: Expose "clearAbortCause" for don’t pick up handled persistence Exceptions

Bug

  • ISIS-1525 - Problems with startup under certain circumstances

  • ISIS-1524 - Dates not showing in view model

  • ISIS-1520 - Amelia template is broken

  • ISIS-1517 - Typo on home page

  • ISIS-1516 - Typo on home page

  • ISIS-1515 - BackgroundCommandExecution can NPE if it attempts to invoke an action on an object that no longer exists.

  • ISIS-1502 - Error handling of invalid domain object not strong enough, causing infinite loop in endTransaction.

  • ISIS-1498 - BackgroundCommandExecution does not handle exceptions correctly.

  • ISIS-1497 - AbstractIsisSessionTemplate does not lookup adapter for OidDtos correctly.

  • ISIS-1494 - EmailNotificationServiceDefault does not use overriding implementations of EmailService

  • ISIS-1493 - Sorting of records with null values is not working correctly

  • ISIS-1490 - Fix 'svg' suffix in ImageResourceCacheClassPath

  • ISIS-1483 - Tiny bug in the SimpleObjectTest unit test as generated by the simpleapp.

  • ISIS-1482 - Occasional(?) exception in i18n PoWriter, looks like a concurrent access problem

  • ISIS-1481 - Duplicate PK exception for SessoinLogEntry (isisaddons implementation of SessionLoggingService) due to race condition.

  • ISIS-1480 - LocalDateTime property on different tabs, encoding exception

  • ISIS-1477 - Background command execution of commands with 0 parameters causes an NPE

  • ISIS-1475 - Application name not set

  • ISIS-1474 - Cannot use custom implementation of authenticationmanager (ShiroWithOAuth2AuthenticationManagerInstaller)

  • ISIS-1472 - Any attempts to schedule a background job via the RO viewer fail, due to incomplete state. (In fact: actions with commands cannot be invoked via RO viewer).

  • ISIS-1470 - Maintenance 1.13.1 umbrella ticket
