1.13.2

This is a maintenance release with a number of minor improvements and bug fixes. If these the main improvement is that the simpleapp archetype has been substantially reworked to be more modular.

This release also closes the ticket to provide a better DB migration story; this has been implemented by way of the (non-ASF) Incode Platform's flywaydb module.

New Feature

  • ISIS-1542 - Update simpleapp application/archetype to provide access to HSQL DB Manager (via their Swing UI) when running in-memory

  • ISIS-1471 - Integrate Flyway or Liquibase or something similar (see (non-ASF) Incode Platform's flywaydb module.

Improvement

  • ISIS-1554 - Prevent double click on buttons (which can cause applications to throw an error, eg concurrency exception)

  • ISIS-1553 - If in prototyping mode, shouldn’t have to keep logging in when restart (rememberMe should activate)

  • ISIS-1552 - Use 'readonly' rather than 'disabled' tag, in compliance with w3c spec (Firefox and Chrome 54+)

  • ISIS-1550 - Remove metamodel validation that insists on at least one @PersistenceCapable class.

  • ISIS-1548 - Allow @Mixin annotation to specify the method name to use (instead of '$$') meaning to infer the mixin name from the mixin type. Also, allow '$' to be used as separator of the mixin type.

  • ISIS-1546 - Reduce logging (don’t show stack traces) for invalid user/pass attempts

  • ISIS-1543 - IsisConfigurationBuilder should obfuscate password when logged.

  • ISIS-1538 - Rework simpleapp archetype so that is more modular.

  • ISIS-1503 - Allow security checking be disabled when running via SudoService.

Bug

  • ISIS-1545 - QueryResultsCache can sometimes result in throwing an exception if not properly initialized when fixtures being installed.

  • ISIS-1541 - When no explicit layout xml is available the order of the members is set to incorrect, ignoring other facets

  • ISIS-1533 - Fix edit icon offset for boolean, referencem blob, clob and multilinestring panels

  • ISIS-1524 - Dates not showing in view model

  • ISIS-1496 - A domainobject annotated with @DomainObject(auditing = Auditing.DISABLED) is audited nonetheless

Task

  • ISIS-1540 - Release tasks for 1.13.2
