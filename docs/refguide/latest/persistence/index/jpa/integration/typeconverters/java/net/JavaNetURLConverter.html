JavaNetURLConverter

Maps an URL to a simple String column in the corresponding database table.

Without this conversion, JPA maps Serializable types to a byte array.

API

JavaNetURLConverter.java
class JavaNetURLConverter {
  String convertToDatabaseColumn(URL uuid)
  URL convertToEntityAttribute(String datastoreValue)
}
