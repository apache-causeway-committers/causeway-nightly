Articles, Conference Sessions, Podcasts

Some articles and recorded material relating to either Apache Isis (or its predecessor, the Naked Objects framework), or its 'sister' open source project on the .NET platform, Naked Objects .NET. We’ve highlighted some of the better ones…

2019

  • µCon London 2019: Leveraging Metamodels and DCI to build Modular Monoliths

2018

  • NCrafts 2018: Petclinic Tutorial

2017

  • Presentation, IMWorld 2017, Bucharest: Thinking outside the box - why you should build, not buy (45 min video)

  • Article, infoq.com: "In Defence of the Monolith" (part 1 and part 2), also re-published in an e-book.

2016

  • presentation, JAX London: Apache Isis: closing the architecture/code gap. Not videotaped, but slides/demos available here.

  • presentation/live coding, Skillsmatter: Closing the Feedback loop with Apache Isis

  • tutorial, SPA 2016: "Naked Objects, 14 years on". Slides available here

2014

  • Presentation, JEEConf: Extremely rapid application development with Apache Isis (50 min. video)

  • Article, SDJournal: Restful Objects on Apache Isis

  • A Prezi presentation on Naked Objects and Apache Isis, mixed by Thomas Eck

  • Article, Linux Magzin (german): Domain-Driven Design mit Apache Isis by Frank Pientka

  • Article, InfoQ: Introducing Restful Objects

2010

2009
